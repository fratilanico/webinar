## What We Are Building

We are building a web app where you fill in your work experience and skills, then click a button and AI rewrites it to sound amazing. You end up with a polished, professional CV you can share with anyone — live on the internet, for free.

> 📸 *Screenshot coming soon*

## The Tools We Use

You do not need to know these already. We will set them up together.

- **Next.js** — the framework that powers our app (think of it as the skeleton of the website)
- **Supabase** — your database in the cloud (where your CV data gets saved)
- **Claude AI** — the brain that reads your CV and rewrites it to sound great
- **Vercel** — where we put the app on the internet so anyone can see it

## What We Will Do Together

1. **Set up the project** — create a fresh app and install everything we need
2. **Build the CV form** — add fields for your name, experience, education, and skills
3. **Connect the database** — save your CV data to Supabase so it persists
4. **Add AI magic** — wire up Claude AI so it can improve your CV text with one click
5. **Make it look great** — add styling so the app looks clean and professional
6. **Deploy it live** — push everything to Vercel so your CV builder is on the internet
