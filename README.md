# APEX OS Live Webinar — AI-Powered CV Builder

We are building a web app where anyone can create a professional CV, get AI suggestions to improve it, save it to the cloud, and share a live link — all in one session.

**Live repo:** https://github.com/fratilanico/webinar
**Code repo:** https://github.com/fratilanico/cv-builder

---

## THE TOOLS

- **Next.js** — the app we build in
- **Supabase** — free cloud database
- **Claude AI** — the brain that writes CV text
- **Vercel** — where we put it live online
- **shadcn/ui** — ready-made buttons and forms

---

## BEFORE THE WEBINAR

Make sure you have these ready before the session starts:

- Install Node.js on your computer (download from nodejs.org, free)
- Create a free Supabase account at supabase.com
- Create a free Anthropic account at console.anthropic.com
- Create a free Vercel account at vercel.com
- Create a GitHub account at github.com

---

## WHAT WE BUILD TOGETHER

1. Set up the project — create the app, install the tools, connect to the database
2. Build the CV form — personal info, work experience, education, and skills sections
3. Add a live preview — the CV updates on screen as you type
4. Connect Claude AI — add a button that rewrites each section to sound more professional
5. Save to the cloud — store CVs in Supabase so users can come back and edit later
6. Deploy live — push the app to Vercel so it works on a real URL anyone can visit
